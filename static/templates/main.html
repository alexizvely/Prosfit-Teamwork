Why use SVG?

SVG is a file format that enables two-dimensional images to be displayed in XML on the Web. The PNG’s and GIF’s that we have been used to using day in and day out are getting ever more old and dated, this is especially true when you start to look at responsive sites as we are finding ourselves doing more and more these days.
